# Contributing

Contributions are welcome! Please open an issue or pull request.

## Development

```bash
git clone https://github.com/solarluminea/envcheck
cd envcheck
node test.js
```

## Submitting changes

1. Fork the repository
2. Create a branch: `git checkout -b my-feature`
3. Make changes and run tests: `node test.js`
4. Open a pull request
